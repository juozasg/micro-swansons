    class NATS::Subscription

    • NATS::Subscription

    Overview

    A subscription to a given subject and possible queue group. Used to unsubscribe/close.

    nc = NATS::Connection.new("demo.nats.io")
sub = nc.subscribe("foo") { |msg| }
sub.close
sub.unsubscribe
